# Componentes de UI

Esta página demonstra os componentes de UI disponíveis para uso na documentação dos Agentes de Conversão.

## Tabs

<Tabs>
  <Tab title="Agentes">
    Os agentes são assistentes virtuais que podem ser configurados para automatizar conversas
    e auxiliar usuários em tarefas específicas, como vendas, suporte ou atendimento.
  </Tab>
  <Tab title="Conversas">
    As conversas representam interações entre usuários e agentes. Cada conversa possui
    um histórico de mensagens e metadados associados.
  </Tab>
  <Tab title="Base de Conhecimento">
    A base de conhecimento fornece informações que ajudam os agentes a responder perguntas
    específicas do seu negócio ou domínio.
  </Tab>
  <Tab title="Analytics">
    O módulo de analytics oferece insights sobre o desempenho dos seus agentes,
    incluindo métricas de engajamento e conversão.
  </Tab>
</Tabs>

### Exemplo de autenticação

```bash
curl -X POST https://api.agentesdeconversao.com.br/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "seu-email@exemplo.com",
    "password": "sua-senha"
  }'
```

## Accordion

<AccordionGroup>
  <Accordion title="Como funciona a integração com WhatsApp?">
    A integração com WhatsApp é feita através do Z-API, que serve como ponte entre a plataforma
    Agentes de Conversão e o WhatsApp Business. Você precisa conectar sua conta Z-API, configurar
    os webhooks necessários e definir as regras de encaminhamento de mensagens.
  </Accordion>
  <Accordion title="Posso personalizar as respostas do meu agente?">
    Sim, você tem controle total sobre a personalidade e comportamento do seu agente. Isso inclui
    o tom de voz, nível de formalidade, conhecimento especializado e fluxos de conversação.
  </Accordion>
  <Accordion title="Como adicionar uma base de conhecimento?">
    Você pode adicionar documentos (PDF, Word, Excel), URLs de websites, ou inserir informações
    diretamente na plataforma. O sistema processará essas informações e as tornará disponíveis
    para consulta pelos seus agentes durante as conversas.
  </Accordion>
  <Accordion title="Quais relatórios de analytics estão disponíveis?">
    A plataforma oferece relatórios detalhados sobre volume de mensagens, tempo de resposta,
    satisfação do cliente, tópicos mais discutidos e taxas de conversão. Todos os relatórios
    podem ser filtrados por período, agente ou canal.
  </Accordion>
</AccordionGroup>

## API Method

<ParamField path="phone" type="string" required>
  Número de telefone no formato internacional (ex: 5511999999999)
</ParamField>

<ParamField path="message" type="string" required>
  Texto da mensagem a ser enviada
</ParamField>

<ParamField path="media_url" type="string">
  URL de mídia para enviar junto com a mensagem (opcional)
</ParamField>

<ParamField path="delay" type="number">
  Tempo de espera em segundos antes de enviar a mensagem
</ParamField>

<RequestExample>
```bash
curl -X POST https://api.agentesdeconversao.com.br/whatsapp/send \
  -H "Authorization: Bearer {seu_token}" \
  -H "Content-Type: application/json" \
  -d '{
    "phone": "5511999999999",
    "message": "Olá! Como posso ajudar hoje?",
    "delay": 2
  }'
```
</RequestExample>

<ResponseExample>
```json
{
  "success": true,
  "message_id": "msg_123456789",
  "status": "queued",
  "sent_at": "2023-06-15T14:30:45Z"
}
```
</ResponseExample>
