## Categorias Principais

A API dos Agentes de Conversão oferece várias categorias de endpoints:

### WhatsApp
- Envio de mensagens
- Encaminhamento de mensagens
- Manipulação de webhooks

### Banco de Dados (Supabase)
- Consulta de dados
- Inserção de dados
- Atualização de dados
- Exclusão de dados

### Autenticação
- Cadastro
- Login
- Redefinição de senha

### Serviços de IA
- Completions de chat
- Gerenciamento de conversas

## Suporte a Múltiplos Provedores de IA

A API suporta vários provedores de IA, incluindo:
- OpenAI
- Anthropic
- Outros modelos configuráveis

## Tratamento de Erros

A API retorna códigos de status HTTP padrão e mensagens de erro detalhadas para indicar problemas com solicitações. Consulte a documentação de cada endpoint para obter informações específicas sobre possíveis erros e como lidar com eles.
